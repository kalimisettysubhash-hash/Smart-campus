# Smart Campus Portal

A React + Vite student portal for campus workflows.

## Features

- Login, registration, protected routes, and password reset
- Student dashboard with schedule, notices, calendar, services, and recent activity
- Academic Hub for notes, bookmarks, upvotes, sharing, preview, and PDF links
- Campus Store marketplace with cart, local order checkout, seller details, and listings
- Issue Reporter for submitting and tracking campus support tickets
- Events page with category filters and RSVP tracking
- Editable student profile with saved profile data and password change

## Tech Stack

- React
- Vite
- React Router
- LocalStorage for demo persistence

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal, usually:

```text
http://localhost:5173
```

## Build

```bash
npm run build
```

## Notes

This project stores demo users, profile details, notes, cart items, orders, and tickets in browser localStorage.
